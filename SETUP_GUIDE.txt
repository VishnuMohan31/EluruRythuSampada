🚀 TRIBES INDIA MARKETPLACE - SETUP GUIDE
==========================================

FRONTEND SETUP (Complete UI - Ready to View!)
----------------------------------------------

1. Navigate to frontend directory:
   cd frontend

2. Install dependencies:
   npm install

3. Create .env file:
   cp .env.example .env

4. Start development server:
   npm run dev

5. Open browser:
   http://localhost:3000
